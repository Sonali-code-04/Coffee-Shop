# ☕ Coffee Shop Website

## 🚀 Live Demo

https://coffee-shop-rouge-rho.vercel.app/


## 📌 Description

A modern and responsive coffee shop website built using HTML, CSS, and JavaScript.
The website showcases a complete café experience with multiple sections including menu, testimonials, gallery, and contact information.


## ✨ Features

### 🏠 Home Page

* Attractive landing section
* Highlights café branding and offerings
* Smooth navigation to other sections


### 📖 About Section

* Information about the coffee shop
* Story, mission, or background of the café


### 📋 Menu Section

* Display of available food and beverages
* Organized layout for easy viewing


### 💬 Testimonials

* Customer reviews and feedback
* Builds trust and authenticity


### 🖼️ Gallery

* Visual showcase of café ambiance and products
* Enhances user experience


### 📞 Contact Section

* Contact information (phone, email, location)
* Allows users to connect easily


### 📱 Responsive Design

* Fully responsive across mobile, tablet, and desktop
* Clean and modern UI


## 🛠️ Tech Stack

* HTML
* CSS
* JavaScript


## 📂 Project Structure

* Structured sections for each page/feature
* Organized layout for easy navigation
* Clean and readable code
