# ☕ Coffee Shop Website

## Description

A responsive coffee shop website built using HTML, CSS, and JavaScript.
This project showcases a modern UI design for a café, including sections like menu, services, and contact.


## Features

* 🏠 Home page with attractive layout
* 📋 Menu section displaying products
* 📞 Contact section
* 📱 Responsive design for different devices
* 🎨 Clean and modern UI


## Technologies Used

* HTML
* CSS
* JavaScript


## Learning Outcome

This project helped me understand:

* Website layout design
* Responsive design techniques
* Styling using CSS
* Structuring multi-section webpages


## How to Run

1. Download or clone the repository
2. Open `index.html` in your browser
